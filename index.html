SUMMARY: LATITUDE VS. X

The purpose of this project was to analyze how weather around the world changes and how it can relate or diffentiate depending on where the city is from the equator. To achieve this, a Python script was created to visualize the weather of 500+ cities of varying distance from the equator using a simple Python library, and OpenWeatherMap API

We needed to create a series of scatter plots to showcase the following relationships:
      • Temperature (F) vs. Latitude
      • Humidity (%) vs. Latitude
      • Cloudiness (%) vs. Latitude
      • Wind Speed (mph) vs. Latitude

To create these scatter plots, Matplotlib was used. This site shows the data and visualizations that were created to show the analysis, and also has descriptions to each of the visualizations that were made.
